## What is Toon Format?

Toon format is a structured representation of JSON data optimized for visualization and processing. This tool helps you convert your JSON data into toon format quickly and efficiently.

## Setup

Make sure to install dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run dev
```

## Usage

1. Enter your JSON data in the input field
2. Click "Convert to Toon" button
3. Copy the converted toon output
4. Use the "Load Example" button to see a sample conversion
